## Amazon Delivery Operations Analytics

This project focuses on analyzing last-mile delivery operations using a publicly available dataset from Kaggle. The main objective was to understand the key factors that cause delivery delays and transform raw operational data into meaningful business insights.

## Tools Used
The analysis was carried out using the following tools:

- **Python** (Pandas, NumPy, Matplotlib) for data cleaning and preprocessing  
- **SQL Server** for structured data analysis and querying  
- **Power BI** for building interactive dashboards and visualizations  
- **Excel** for initial data inspection and understanding the dataset structure  

## Project Workflow
The project followed a structured data analytics workflow:

1. **Data Cleaning and Preparation** – The dataset was cleaned and processed using Python to handle missing values, remove incorrect records, and create useful features.
2. **SQL Analysis** – The cleaned dataset was imported into SQL Server to analyze delivery performance and identify operational patterns.
3. **Visualization** – A Power BI dashboard was created to present the insights in a clear and interactive format for business decision-making.

## Key Insights
The analysis revealed several important factors affecting delivery performance:

- Delivery time increases significantly during **high traffic conditions**.
- **Longer delivery distances** lead to longer delivery times.
- **Adverse weather conditions** can slow down delivery operations.
- Assigning **multiple deliveries to a single delivery partner** increases the likelihood of delays.

## Dashboard
An interactive Power BI dashboard was developed to visualize key performance indicators such as total orders, average delivery time, delivery distance, and delivery partner ratings. The dashboard also highlights the impact of traffic, weather conditions, and workload on delivery performance.

## Dataset
The dataset used in this project was obtained from **Kaggle** and represents simulated delivery operations similar to those used by large logistics and e-commerce companies.
